
# Expense Tracker (Full Stack)

A production-ready expense tracking app with authentication, analytics, and AI-powered financial insights.

## Tech Stack
- Next.js (App Router)
- Prisma ORM
- PostgreSQL (Neon)
- NextAuth (Credentials)
- OpenAI API
- TailwindCSS
- Vercel Deployment

## Features
- Secure user authentication
- Add / edit / delete expenses
- Category filtering + search
- Monthly analytics & charts
- AI-generated financial insights
